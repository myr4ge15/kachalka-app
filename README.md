# kachalka-app

Приложение для закрытого круга друзей, которые ходят в зал, следят за прогрессом в силовых и немного соревнуются. Открывается в браузере телефона и ставится на рабочий стол как обычное приложение.

## Зачем это нужно

Заметки и таблицы не дают увидеть динамику и сравнить себя с друзьями. Здесь всё заточено под одно: быстро записать подход прямо в зале и увидеть прогресс.

- ⚡ **Быстрый ввод** — упражнение и подходы в несколько тапов крупными кнопками.
- 📈 **Виден прогресс** — график силы по любому упражнению, рекорды отмечаются автоматически.
- 🏆 **Соревнование** — общая лента и рейтинг по жиму.

Круг закрытый, все видят результаты друг друга. Администратор может сделать участника **приватным** (его тренировки, рейтинг и цели скрыты ото всех, кроме него и админа). При этом администратор может открыть приватному **взаимный доступ** с избранными людьми — они видят тренировки друг друга, но остальным приватный по-прежнему не виден и в общий рейтинг не попадает.

## Что внутри 📱

Внизу — панель вкладок с иконками; повторный тап по открытой вкладке прокручивает её наверх. На широком экране (компьютер, планшет) эта панель переезжает влево боковой колонкой (профиль, статус синхронизации и уведомления — внизу неё, шапка убирается), а содержимое раскрывается шире — приложение перестаёт выглядеть как растянутый телефон.

- 🏠 **Главная** — открывается сразу при входе. Последняя тренировка, серия недель 🔥, 2–3 **авто-вывода** (новый рекорд, обгон друга по жиму, рекордный объём, забытая группа, тренд тоннажа, серия), цифры месяца с дельтой и ближайшая цель с прогресс-баром. Всё считается на устройстве, работает офлайн.
- 💪 **Мои тренировки** — прошлые тренировки карточками с редактированием. «+ Добавить тренировку» открывает композер с общим справочником (поиск, фильтр по группам, своё упражнение прямо из поиска с подсказкой похожих). Упражнение можно **заменить** без повторного ввода подходов. Знакомое упражнение подставляется **рекомендацией на сегодня** («всё выполнено → +2.5 кг», «не добил → тот же вес», «тяжело → снизим») с кнопкой «вернуть как в прошлый раз» и настройкой стратегии (+вес / +повторы / ручной / выкл) на шестерёнке; отключается тумблером в Профиле. Три типа упражнений: «вес × повторы», «только повторы» и «на время» (ввод мм:сс). Карточки сами получают **теги дня** по группам, сверху — фильтр. Тренировки можно **выгрузить в JSON**, начатую — **очистить** одним тапом. На широком экране список и редактор рядом.
- 📋 **Шаблоны** — комплекс применяется одним тапом; для каждого упражнения задаются целевые **подходы × повторы × вес**. Готовую тренировку можно превратить в шаблон одной кнопкой («Сделать шаблон из тренировки» — план берётся по лучшему подходу каждого упражнения). Можно сделать общим для круга. Разбиты на свёрнутые группы «Мои» и «Общие»; выгрузка в JSON поштучно или пачкой.
- 🔥 **Лента** — тренировки всех участников и рейтинг (по факт. макс. весу, 1ПМ сноской), **раздельный по полу**: парни по жиму, девушки по ягодичному мостику. Рекорд — кубком 🏆. Под тренировкой друга — **реакции** (💪/🔥/👏/😮) со счётчиком и именами; работают офлайн.
- 📊 **Прогресс** — график силы и таблица по дням для любого упражнения (рост зелёным, спад красным, рекорды жёлтым). Период: Неделя · Месяц · Всё · Произвольный.
- 🔔 **Уведомления** — новый рекорд, побитый рекорд, реакции на твою тренировку и авто-выводы (те же, что на Главной), с фильтром по категориям. Считается на устройстве, офлайн.
- 🤖 **Telegram-бот** — поздравления с рекордом и целью плюс вечерняя сводка дня в общий чат.
- 👤 **Профиль** (тап по имени) — аватар (в т.ч. HEIC), смена имени, статистика за всё время (тренировки и **тоннаж**), **личные цели** по весу/повторам/времени с прогресс-баром, личные рекорды, смена PIN, удаление данных, выход. Здесь же **«Мои упражнения»** — список добавленных участниками упражнений с редактированием (название, тип, группа и мышцы); правки видны всем.
- 🏆 **Достижения** (из Профиля) — бейджи за регулярность (10/50/100 тренировок), серии недель подряд, суммарный тоннаж (10/100/1000 тонн) и число личных рекордов. У полученных — дата, у остальных — прогресс до следующей вехи; новая веха после тренировки даёт поздравление. Считается на устройстве из твоей истории, офлайн.

## Как это работает ⚙️

**🔑 Вход по PIN.** Регистрации нет: выбираешь себя из списка и вводишь четырёхзначный PIN. Это простой вариант для доверенного круга, а не строгая защита. PIN хранится на сервере только как хэш (PBKDF2 с солью) и проверяется на сервере, после чего приложение получает защищённую сессию. Без сети вход тоже работает — PIN сверяется с локальной копией хэша. Сменить PIN можно в «Профиле» (нужна сеть).

**📴 Работает офлайн.** Данные лежат на телефоне, экраны показывают локальную копию — открываются мгновенно и без интернета. Новая тренировка сохраняется сразу, отправка ставится в очередь. На общем телефоне у каждого своя копия: после входа под другим именем чужого не видно.

**🔄 Синхронизация в фоне.** Как появляется сеть, приложение само досылает очередь и подтягивает тренировки друзей. В шапке — значок статуса (синхронизировано / идёт / в очереди / офлайн / ⚠ ошибка). Повторная отправка не задваивается; правку одной тренировки с двух устройств разрешает более поздняя версия, а перезаписанная попадает в журнал, а не теряется молча. Синхронизация **инкрементальная** (только изменения с прошлого раза), а лента и рейтинг обновляются **вживую** — рекорд друга виден почти сразу.

**⬆️ Обновления.** Приложение закешировано; новая версия тихо скачивается в фоне, внизу над меню появляется компактная пилюля «Новая версия» с кнопкой «Обновить». Крестик рядом откладывает её — вернётся сама позже, если обновление ещё ждёт. Зависшие изменения можно дослать вручную из «Профиля».

**🌐 Общий сервер.** Сайт и иконка на телефоне — один веб-клиент, данные синхронизируются через общий сервер. Свои тренировки можно менять, чужие — только смотреть.

**💪 Свежесть по группам мышц.** На Главной — карточка «Свежесть групп»: цветная полоска показывает, восстановилась ли группа (красный — дай отдых, жёлтый — почти, бирюзовый — можно), а подпись под ней говорит то же самое словами: пора кого-то проработать, кто-то ещё отдыхает или все мышцы свежие. Тап открывает подробный экран с картой тела (два силуэта спереди/сзади, раскрашены по свежести — тап по мышце подсвечивает её): по каждой группе видно, сколько дней назад её тренировали и можно ли уже снова, плюс подсказка про перекос (например, две недели без ног). В списке «когда снова тренировать» можно тапнуть строку — экран сам прокрутится к карте тела и подсветит эту мышцу. **Кор** (планка и другие стабилизаторы) теперь отслеживается отдельно от **пресса** (кранчи, скручивания) — их свежесть больше не путается. Всё считается на телефоне из твоих тренировок и работает офлайн.

## Как пользоваться 🚀

1. Открой приложение и добавь его на главный экран телефона.
2. На входе выбери себя и введи PIN.
3. В **Моих тренировках** жми «+ Добавить тренировку», добавляй упражнения по ходу занятия, в конце — «Сохранить». Частый набор сохрани в **Шаблоны**.
4. На **Главной** сразу видно сводку и авто-выводы. Динамику смотри в **Прогрессе**, тренировки друзей и рейтинг — в **Ленте**, настройки и цели — в **Профиле**.
