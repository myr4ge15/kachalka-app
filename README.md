# kachalka-app

Приложение для закрытого круга друзей, которые ходят в зал, следят за прогрессом в силовых и немного соревнуются. Открывается в браузере телефона и ставится на рабочий стол как обычное приложение.

## Зачем это нужно

Заметки и таблицы не дают увидеть динамику и сравнить себя с друзьями. Здесь всё заточено под одно: быстро записать подход прямо в зале и увидеть прогресс.

- ⚡ **Быстрый ввод** — упражнение и подходы в несколько тапов крупными кнопками.
- 📈 **Виден прогресс** — график силы по любому упражнению, рекорды отмечаются автоматически.
- 🏆 **Соревнование** — общая лента и рейтинг по жиму.

Круг закрытый, все видят результаты друг друга. Администратор может сделать участника **приватным** (его тренировки, рейтинг и цели скрыты ото всех, кроме него и админа).

## Что внутри 📱

Внизу — панель вкладок с иконками; повторный тап по открытой вкладке прокручивает её наверх. На широком экране (компьютер, планшет) эта панель переезжает влево боковой колонкой (профиль — внизу неё), а содержимое раскрывается шире — приложение перестаёт выглядеть как растянутый телефон.

- 💪 **Мои тренировки** — главный экран: прошлые тренировки карточками с редактированием. «+ Добавить тренировку» открывает композер с общим справочником упражнений (поиск, фильтр по группам мышц, можно завести своё прямо из поиска — с подсказкой похожих). Любое упражнение можно **заменить**, не вводя подходы заново. Знакомое упражнение подставляется с **подходами из прошлой тренировки** по нему — не набирать вес и повторы вручную. Три типа упражнений: «вес × повторы», «только повторы» (подтягивания, пресс) и «на время» (планка, кардио — ввод в мм:сс). Карточки сами помечаются **тегами дня** по группам мышц, сверху — фильтр. На широком экране список и открытая тренировка показываются рядом (список слева, редактор справа). Тренировки можно **выгрузить в JSON** — поштучно или пачкой. Начатую новую тренировку легко **очистить** одним тапом, если передумал её сохранять (в т.ч. набранный из шаблона состав).
- 📋 **Шаблоны** — заранее собранный комплекс применяется одним тапом. Для каждого упражнения задаются целевые **подходы × повторы × вес**, которые подставляются готовыми. Шаблон можно сделать общим для круга.
- 🔥 **Лента** — последние тренировки всех участников плюс рейтинг (по факт. макс. весу; 1ПМ — сноской). Рейтинг **раздельный по полу**: парни — по жиму, девушки — по ягодичному мостику; каждый видит свой борд. Новый рекорд — значком кубка 🏆. Под тренировкой друга можно поставить **реакцию** (💪/🔥/👏/😮) — со счётчиком и именами отметившихся; работает и офлайн. Под своей тренировкой видно, кто отметил, но самому реагировать нельзя.
- 📊 **Прогресс** — график силы и таблица по дням для любого упражнения. Рост — зелёным, спад — красным, рекорды — жёлтым. Период: Неделя · Месяц · Всё · Период. На широком экране выбор упражнения и период стоят сбоку, а график и таблица — рядом.
- 🔔 **Уведомления** — «новый рекорд» и «друг побил твой рекорд». Считается на устройстве, работает офлайн.
- 🤖 **Telegram-бот** — поздравления с рекордом и целью плюс вечерняя сводка дня в общий чат.
- 👤 **Профиль** (тап по имени в шапке) — аватар (в т.ч. HEIC с iPhone), смена имени, статистика (тренировок всего и за месяц, **серия недель подряд** 🔥 и суммарный **тоннаж**), **личные цели** по весу/повторам/времени с прогресс-баром (у весовой можно задать и повторы при целевом весе), личные рекорды, смена PIN, удаление своих данных, выход.

## Как это работает ⚙️

**🔑 Вход по PIN.** Регистрации нет: выбираешь себя из списка и вводишь четырёхзначный PIN. Это простой вариант для доверенного круга, а не строгая защита. PIN хранится на сервере только как хэш (PBKDF2 с солью) и проверяется на сервере, после чего приложение получает защищённую сессию. Без сети вход тоже работает — PIN сверяется с локальной копией хэша. Сменить PIN можно в «Профиле» (нужна сеть).

**📴 Работает офлайн.** Все данные лежат на телефоне, экраны показывают локальную копию — открываются мгновенно и без интернета. Новая тренировка сохраняется сразу, отправка ставится в очередь. На общем телефоне у каждого участника своя локальная копия данных — после выхода и входа под другим именем чужих тренировок, ленты и уведомлений не видно.

**🔄 Синхронизация в фоне.** Как появляется сеть, приложение само досылает очередь и подтягивает тренировки друзей. В шапке — компактный значок статуса: галочка (всё синхронизировано), крутящийся кружок (идёт синхронизация), стрелка с числом (что-то в очереди), перечёркнутое облако (офлайн) или ⚠ (что-то не отправилось и нужно дослать вручную — либо последняя синхронизация не прошла из-за связи). Значок больше не показывает «синхронизировано», когда обновление на самом деле сорвалось по сети. Повторная отправка не задваивается. Если одну тренировку правили с двух устройств, выживает более поздняя версия, а перезаписанная не теряется молча — попадает в журнал и отмечается в статусе синхронизации. Синхронизация **инкрементальная**: подтягиваются только изменения с прошлого раза, а не вся база каждый раз — меньше расхода батареи и трафика на телефоне. А ещё лента и рейтинг обновляются **вживую**: как только друг сохранил тренировку или побил рекорд, это видно почти сразу, без ожидания — и без постоянной фоновой нагрузки на телефон.

**⬆️ Обновления.** Приложение закешировано; новая версия тихо скачивается в фоне, внизу над меню появляется компактная пилюля «Новая версия» с кнопкой «Обновить». Крестик рядом откладывает её — вернётся сама позже, если обновление ещё ждёт. Зависшие изменения можно дослать вручную из «Профиля».

**🌐 Общий сервер.** Сайт и иконка на телефоне — один веб-клиент, данные синхронизируются через общий сервер. Свои тренировки можно менять, чужие — только смотреть.

## Как пользоваться 🚀

1. Открой приложение и добавь его на главный экран телефона.
2. На входе выбери себя и введи PIN.
3. В **Моих тренировках** жми «+ Добавить тренировку», добавляй упражнения по ходу занятия, в конце — «Сохранить». Частый набор сохрани в **Шаблоны**.
4. Смотри динамику в **Прогрессе**, тренировки друзей и рейтинг — в **Ленте**, сводку по себе — в **Профиле**.
